## Chrome Shortcuts

**Focus Input**  
`Cmd + l`

**Open New Tab**  
`Cmd + t`

**Close Current Tab**  
`Cmd + w`

**New Window**  
`Cmd + n`

**Close Session**  
`Cmd + Shift + w`

**Restore Session**
`Cmd + Shift + t`

**Reopen Closed Tab**  
`Cmd + Shift + t`

**Cycle Tabs**  
`Cmd + Shift + [ or ]`

**History**  
`Cmd + y`

**Back**  
`Cmd + ←`

**Forward**  
`Cmd + →`

**Page Down**  
`Space bar`

**Page Up**  
`Shift + space bar`

**Focus Search Input**  
`Cmd + option + f`
