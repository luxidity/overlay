`Shift + Cmd + N`

`Shift + Cmd + L`

`Cmd + Del`

`Cmd + B`

`Shift + Cmd + F`

`Cmd + +` / `Cmd + -`

`Cmd + Option + F`

`Cmd + K`

`Cmd + Option + S`

`Cmd + 1`
