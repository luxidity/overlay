## VS Code Shortcuts

### Main Workflow

**Bread Crumb**  
`Shift + Cmd + .`

**Format Selection**  
`Cmd + K + Cmd + F`

**Format Document**  
`Shift + Option + F`

**Open Recently Used Files**  
`Cmd + P`

**Run Any Function**  
`Cmd + P + >`

**Cycle Open Workspaces**  
`Cmd + \``

**Go Back**  
`Ctrl + -`

**Go Forward**  
`Ctrl + =`

**Next Tab**  
`Ctrl + Tab`

**Previous Tab**  
`Ctrl + Shift + Tab`

**Jump to Line**  
`Ctrl + G`

**Forward/Back**  
`Shift + Cmd + [` / `Shift + Cmd + ]`

**Copilot Inline Chat**  
`Cmd + I`
